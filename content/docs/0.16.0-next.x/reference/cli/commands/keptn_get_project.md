## keptn get project

Get details of a Keptn project

### Synopsis

Get all projects or details of a given Keptn project

```
keptn get project [flags]
```

### Examples

```
keptn get projects

sockshop        2020-05-28T10:25:50+02:00

keptn get project sockshop
```
